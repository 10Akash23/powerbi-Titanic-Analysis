# powerbi-Titanic-Analysis
This project presents an interactive data visualization dashboard created using Microsoft Power BI to analyze the Titanic passenger dataset. The dashboard provides insights into passenger survival patterns based on different factors such as age, gender, passenger class, and fare.

**Objectives**

Analyze survival rate of passengers on the Titanic.
Understand the impact of gender on survival.
Study survival distribution across passenger classes.
Explore age group patterns among survivors and non-survivors.

**Tools Used**

Microsoft Power BI
Microsoft Excel / CSV Dataset
GitHub for project hosting

**Dataset**

The dataset used in this project is the Titanic dataset which contains passenger details such as:

PassengerId
Name
Age
Gender
Passenger Class
Fare

**Data Transformation**

The following transformations were performed before creating the dashboard:
Missing age values handled

**Age grouped into categories:**

Child (0–12)
Adult (13–59)
Senior (60+)
Gender column standardized
Survival column converted into readable format (Survived / Not Survived)

**Dashboard Insights**

Key insights generated from the dashboard include:

Female passengers had a higher survival rate compared to males.
Passengers in 1st class had a better chance of survival.
Children had higher survival rates compared to adults.
Most passengers were in the age group 20–40 years.

**Visualizations Included**

The Power BI dashboard includes the following visualizations:
Survival Rate by Gender
Survival Rate by Passenger Class
Age Group Distribution
Fare Distribution
Overall Survival Percentage

**How to Use**

Download the .pbix file from this repository.
Open it using Microsoft Power BI Desktop.
Interact with the dashboard filters and visuals.
